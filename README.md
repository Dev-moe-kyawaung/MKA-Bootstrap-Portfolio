# Moe-Kyaw-Aung-Portfolio

Premium terminal-style portfolio for **Moe Kyaw Aung**, Senior Android Developer.

## Features
- Bootstrap 5 responsive layout
- Multilingual toggle (English / Burmese)
- Dark / light mode
- Animated cursor, particle background, typed hero
- GitHub repo cards
- Lightbox for images/videos
- Contact and newsletter forms
- PWA-ready with service worker

## Development
Open `index.html` in a browser or serve with any static server.

## Deployment
### GitHub Pages
Use `.github/workflows/deploy.yml`.

### Firebase Hosting
```bash
firebase init hosting
firebase deploy
